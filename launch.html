CARA INSTAL

CHROME / EDGE

  1. Download folder ekstensi ini.
  2. Buka chrome://extensions.
  3. Aktifkan Developer Mode di pojok kanan atas.
  4. Klik Load Unpacked.
  5. Pilih folder auto-fill-extension.

FIREFOX

  1. Buka browser Firefox.
  2. Akses about:debugging.
  3. Pilih This Firefox.
  4. Klik Load Temporary Add-on.
  5. Pilih file manifest.json di folder ekstensi.
